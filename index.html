Firebase Echtzeit-Sync

Einmalige Einrichtung – danach synchronisieren alle Geräte automatisch über einen gemeinsamen Raumcode.

1
Firebase-Projekt erstellen Öffne console.firebase.google.com → „Projekt hinzufügen" → beliebiger Name (z. B. „Packit") → Analystics kann deaktiviert bleiben.
2
Firestore-Datenbank aktivieren Im Projekt: „Firestore Database" → „Datenbank erstellen" → Testmodus → Region wählen (z. B. eur3).
3
Sicherheitsregeln setzen In Firestore → Tab „Regeln" → Inhalt ersetzen mit: rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /rooms/{roomCode}/data/{doc=**} { allow read, write: if true; } } } → Veröffentlichen
4
Web-App registrieren & Config kopieren Projektübersicht → ⊕ Web-App → App registrieren → den firebaseConfig-Block kopieren.
